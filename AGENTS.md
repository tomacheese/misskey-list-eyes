# AGENTS.md

## 目的
このドキュメントは、AI エージェント共通の作業方針を定義します。

## 基本方針
- **会話と言語**: 日本語で会話を行い、コード内のコメントも日本語で記載します。
- **エラーメッセージ**: 原則として英語で記載します。
- **コミット規約**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従い、description は日本語で記載します。
- **書式**: 日本語と英数字の間には半角スペースを挿入します。

## 判断記録のルール
重要な設計判断や変更を行った場合は、その理由、検討した代替案、前提条件を記録に残してください。

## 開発手順
1. **プロジェクト理解**: `package.json` や `src/` 配下を確認し、プロジェクトの目的と構造を把握します。
2. **環境構築**: `yarn install` を実行し、依存関係をセットアップします。
3. **実装**: プロジェクトの既存のスタイルを尊重し、docstring を日本語で付与します。
4. **検証**: `yarn lint` を実行し、構文エラーやスタイルの違反がないか確認します。

## セキュリティ / 機密情報
- API キー、アクセストークン、Webhook URL などの機密情報をコードに含めないでください。
- ログに個人情報や機密情報が出力されないよう注意してください。

## リポジトリ固有事項
- このプロジェクトは Misskey API と Puppeteer を使用しています。
- スクリーンショット撮影のためにブラウザが起動されることに留意してください。
